# Claude Code Prompt Training

Tài liệu training 41 slide hướng dẫn viết prompt cho Claude Code CLI — dựa trên dự án thật E2E_TC001.

**Xem live:** https://mor-nhatlq.github.io/claude-code-prompt-training/

## Nội dung

- Phase 1 (slide 6–21): Brainstorm — khám phá phạm vi, duyệt thiết kế, mở feature branch
- Phase 2 (slide 22–32): Live debug — chạy thật trên UAT, gỡ lỗi iframe Razorpay
- Phase 3 (slide 33–38): Polish — review kiểu PR, đẩy ngược thiết kế
- Tổng hợp (slide 39–41): 10 patterns, 5 anti-pattern, cheat sheet
